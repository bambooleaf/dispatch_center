##基础服务任务分发中心

### 接受请求
只接受post请求,具体对应的服务请求类型,写在具体的参数里

 {
 
       "service":"pay",//请求基础服务类型
       "method":"post",//请求的方法类型
       "bizcontent":{
            "a":123
       },//具体业务参数
       "version":1.0,//为以后多版本分类预留
       "module":"pay/postpay"//请求的具体方法
  }

### 返回结果

{

    "status": 0, //错误码,0代表成功
    "msg": "请求成功", //错误信息描述
    "data": { //具体的业务请求返回结果
        "updatetime": "2017-12-12 12:04:36",
        "status": false,
        "ret_code": 200,
        "msg": "验证失败，失败原因：trade_no信息有误"
    }
}

### 错误码
0       成功
40000   请求报文不合法
40001   请求报文校验失败
40004   请求服务不存在
50000   服务方异常,未返回数据
